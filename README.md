Galvenais mērķis ir pabarot būtnes ar barību, kas apmierina viņiem piederošās īpašības(piem, inficēts,korputēts, utt)
Šīs ēdiena attribūtas tiek saglabātas 2d masīvā, kur 1. rinda satur ēdiena nosaukumu, 2. ēdiena vērtību un katra nākamā rinda apzīmēs vai akurātā attribūta pieder šim ēdienam (1 - pieder; 0 - neppieder)

nosaukums:  sula  čebureks

Ja būtne tiek nepareizi pabarota, spēlētājs zaudē dzīvību. Kopā ir 3 dzīvības
